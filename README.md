# 🪢 Hangman

A terminal-based Hangman game built as part of my **100 Days of Code** journey. This project focused on practising **lists, loops, and string manipulation** in Python.

> 📚 Day project from [100 Days of Code: The Complete Python Pro Bootcamp](https://www.udemy.com/course/100-days-of-code/) by Dr. Angela Yu.

## How it works

- Pick a difficulty level, then guess the hidden word one letter at a time
- You have 6 incorrect guesses before the hangman is complete — and you lose
- The ASCII hangman updates in the terminal with each wrong guess

## Difficulty Levels

| Level | Word Length |
|-------|-------------|
| Easy | Up to 5 letters |
| Medium | 6–8 letters |
| Hard | 9+ letters |
| Feeling Lucky | Any word from the full list |

## How to Run

```bash
python main.py
```

No external dependencies — just Python's built-in `random` module.
